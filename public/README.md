# public: Static web-visible resources

Static files used in the project and intended to be publicly accessible. These might include images, icons, fonts, HTML/CSS/JS files, and other media assets.

## What should go in public?

- Favicon, manifest, robots.txt
- Static files served at root (images, stylesheets HTML pages)
- Assets referenced by public URLs

## What should NOT go in public?

- Source files
- Private assets not meant for public access
- Configuration files
